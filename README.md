# LLM-based multi-agent platform

Implementing the entire Docker-based orchestrator + broker + agent + frontend architecture

### Services
- 🧠 Orchestrator (FastAPI)
- 🧭 Broker (FastAPI)
- 🤖 Agent Core / Model (FastAPI)
- 🖥️ Frontend (Next.js with TypeScript)

### How to run
```bash
make build
make up
make logs
```
